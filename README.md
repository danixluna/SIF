# SIF

Sistema de Información Financiera Personal.

SIF es una aplicación enfocada en la administración financiera personal para usuarios avanzados que necesitan comprender no solamente cuánto gastan, sino el contexto completo de cada movimiento financiero.

## Objetivo

Centralizar toda la información financiera del usuario para facilitar la toma de decisiones mediante automatización, organización y análisis.
